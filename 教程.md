 超星学习通自动刷课脚本 — 使用说明

> **重要**：填写的链接必须是课程**视频播放页面**的链接，不是课程首页或任务列表页！

---

## 一、脚本功能

- 自动播放每个知识点的视频，播放完成后自动跳转到下一个知识点
- 视频被暂停时自动恢复播放
- 视频中弹出判断题时自动作答（先选「对」提交，答错自动改选「错」重新提交）
- 自动关闭各类弹窗
- 支持最多 200 个知识点连续刷课

---

## 二、Windows 使用步骤

**1. 解压并修改链接**

将压缩包全部解压，用**记事本**打开文件夹中的 所有python文件`xxx.py`，找到 `URL = (` 那一行，将引号内的链接替换为你自己的课程视频播放页链接，按 `Ctrl + S` 保存。

**2. 打开终端**

在文件夹空白处，按住 `Shift` 并点击鼠标右键，选择【在此处打开 PowerShell 窗口】。

**3. 安装依赖**

```
pip install -r requirements.txt
```

**4. 运行脚本**

```
python auto_study.py
```

浏览器自动弹出后，扫码登录超星账号即可开始自动刷课。

---

## 三、Mac 使用步骤

**1. 解压并修改链接**

将压缩包全部解压，右键 `auto_study.py` 选择【用文本编辑打开】，找到 `URL = (` 那一行，将引号内的链接替换为你自己的课程视频播放页链接，按 `Command + S` 保存。

**2. 打开终端**

在文件夹空白处右键，选择【新建位于文件夹位置的终端窗口】。

**3. 安装依赖**

```bash
pip3 install -r requirements.txt
```

**4. 安装浏览器驱动**（首次使用需要执行）

```bash
python3 -m playwright install
```

**5. 运行脚本**

```bash
python3 auto_study.py
```

浏览器自动弹出后，扫码登录超星账号即可开始自动刷课。

---

## 四、补充说明

- 脚本运行期间**不要关闭**弹出的浏览器窗口和终端窗口
- 电脑可以后台挂机，不影响正常使用其他软件
- 更换课程时，只需重新修改 `auto_study.py` 内的链接，重新运行即可

---

## 五、常见问题

### 1. `ModuleNotFoundError: No module named 'playwright'`

依赖未安装，执行：

```bash
pip3 install playwright --user
```

### 2. `zsh: command not found: playwright`

浏览器驱动未安装，执行：

```bash
python3 -m playwright install
```

### 3. 安装依赖时报 `OSError` 或权限不足

跳过批量安装，单独安装核心依赖：

```bash
python3 -m pip install playwright --user
```

### 4. `pip` 版本过低警告（不影响运行）

可选升级：

```bash
python3 -m pip install --upgrade pip --user
```
