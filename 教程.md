超星自动刷课脚本 使用说明

重要提醒

填写的链接必须是课程视频播放页面链接！

一、Windows 电脑使用步骤

首先把下载好的压缩包全部解压，打开解压出来的文件夹。

在文件夹里找到 auto_study.py 文件，鼠标右键点击该文件，选择打开方式为记事本。

在文件内部找到网址链接位置，删除里面自带的旧链接，替换成你自己的超星视频播放页链接，修改完成后按下 Ctrl + S 保存文件，关闭记事本。

在文件夹空白的地方，按住键盘上的 Shift 键同时点击鼠标右键，选择【在此处打开 PowerShell 窗口】。

在弹出的黑色窗口中，输入下面代码并按下回车，安装运行所需依赖库

```
pip install -r requirements.txt
```

等待依赖安装完成后，继续输入下面代码并回车，启动脚本

```
python auto_study.py
```

脚本运行后会自动弹出浏览器，扫码登录自己的超星账号，即可开始自动刷课。

二、Mac 苹果电脑使用步骤

首先把下载好的压缩包全部解压，打开解压出来的文件夹。

在文件夹里找到 auto_study.py 文件，鼠标右键点击该文件，选择【用文本编辑打开】。

在文件内部找到网址链接位置，删除里面自带的旧链接，替换成你自己的超星视频播放页链接，修改完成后按下 Command + S 保存文件，关闭编辑窗口。

在文件夹空白处点击鼠标右键，选择【新建位于文件夹位置的终端窗口】。

在弹出的终端窗口中，输入下面代码并按下回车，安装运行所需依赖库

```
pip3 install -r requirements.txt
```

等待依赖安装完成后，继续输入下面代码并回车，启动脚本

```
python3 auto_study.py
```

脚本运行后会自动弹出浏览器，扫码登录自己的超星账号，即可开始自动刷课。

补充说明

脚本运行期间不要关闭弹出的浏览器窗口与终端窗口。

电脑可以后台挂机，不影响正常使用电脑其他软件。

每次更换课程，只需要重新修改 auto_study.py 内的视频链接即可。

常见问题：
如果 Mac 用户安装依赖时出现 “权限不足”“无法写入系统目录” 的报错，比如“WARNING: You are using pip version 21.2.4; however, version 26.0.1 is available.
You should consider upgrading via the '/Library/Developer/CommandLineTools/usr/bin/python3 -m pip install --upgrade pip' command.”，请先执行这两行命令配置独立环境：
安装 Homebrew：/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
安装独立 Python：brew install python3
完成后再重新运行pip3 install -r requirements.txt即可解决权限问题。
