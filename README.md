## 脚本功能

这是一个**超星学习通自动刷课脚本**，使用 Playwright 浏览器自动化框架，实现全自动观看网课视频。

### 核心流程

1. **登录** — 自动打开课程页面，等待用户手动扫码或输入账号密码登录（5 分钟内）
2. **点击「2 视频」标签** — 在页面顶部切换到视频标签页，支持重试 6 次
3. **定位视频** — 遍历页面所有 iframe，找到包含 `<video>` 元素的那一层
4. **播放视频** — 通过 JS 调用 `video.play()` 或点击播放按钮
5. **等待播放完成** — 每 5 秒轮询进度，直到 ≥98%
6. **点击右侧列表下一个知识点** — 找到当前高亮项（`.posCatalog_active`），点击其下一个 `span.posCatalog_name`（带 `onclick` 事件的元素）
7. 回到步骤 2，循环

### 辅助功能

| 功能 | 说明 |
|------|------|
| **暂停恢复** | 视频被暂停时自动检测并恢复播放 |
| **答题弹窗** | 视频中弹出判断题时，先选「对」提交；答错则改选「错」再提交；答完自动关闭弹窗 |
| **弹窗处理** | 自动关闭 layui 确认框等普通弹窗 |
| **iframe 失效恢复** | 视频 frame 失效时自动重新定位 |
| **防检测** | 伪装 User-Agent，隐藏 `navigator.webdriver` 标识 |
| **超时保护** | 进度长时间不动（60 次轮询）自动跳过，防止卡死 |
